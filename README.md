 Plataforma Educativa – TFG

Este repositorio tiene el TFG de Nicolás Osete Justicia, consistente en una plataforma educativa creada con ASP.NET MVC. El proyecto incluye gestión de usuarios, visualización de cursos y lecciones, diseño responsive y una interfaz hecha por CSS.

El objetivo principal es ofrecer una plataforma accesible para todos los que quieran aprender y saber algo mas sobre la Informatica.
